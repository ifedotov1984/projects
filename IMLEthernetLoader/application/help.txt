IML_Ethernet_Loader_setup.exe - установочный файл 32-х разрядной версии приложения для Windows
IMLEthernetLoader--x86_64.AppImage* - исполняемый файл 64-х разрядной версии приложения для Linux
IMLEthernetLoader-i386.AppImage* - исполняемый файл 32-х разрядной версии приложения для Linux


*Для запуска исполняемых файлов  *.AppImage необходимо скопировать данный файл в желаемый каталог, 
после чего сделать его исполняемым (выполнить с правами root команду chmod u+x [имя файла]). 
Для запуска приложения с командной строки, перейти в каталог с приложением и выполнить команду ./[имя файла]
